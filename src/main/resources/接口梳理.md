## 用户接口

### 获取手机验证码

#### 接口详情

| 接口详情 | 获取注册验证码     |
| -------- | ------------------ |
| 请求URL  | getOneTimePassword |
| 请求方法 | POST               |

#### 请求参数

| 参数名称     | 参数类型 | 说明   |
| ------------ | -------- | ------ |
| phoneNumber* | String   | 手机号 |

#### 返回值

| 返回字段 | 字段类型 | 说明              |
| -------- | -------- | ----------------- |
| status   | String   | success 、failure |
| data     | T        | 参看接口返回示例  |

#### 接口请求成功示例

```java
//请求URL
http://127.0.0.1:8080/getOneTimePassword
//请求参数
phoneNumber：16621792812
//接口返回值
{
    "status": "success",
    "data": {
        "phoneNumber": "16621792812",
        "otpCode": "5704"
    }
}
```
